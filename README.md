## いいね順並び替え機能、DM機能、閲覧数表示機能

投稿数の表示、前日、前週との比率を表示させる機能
1週間の投稿数を表示させchart.jsを用いて表示、選択した日付の投稿数を表示させる検索機能実装(非同期)

* scopeを用いて昨日、今週、前週期間を記述した
* チャート実装では、turbolinksを無効化して実装しました
* f.date_fieldを用いて、日付選択ができるフィールドを作成
